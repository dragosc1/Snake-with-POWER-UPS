# Snake with POWER-UPS

## Controls:
<kbd>↑</kbd> - move one cell up <br>
<kbd>↓</kbd> - move one cell down <br>
<kbd>←</kbd> - move one cell left <br>
<kbd>→</kbd> - move one cell right
